# Rubber Duck TUI

A simple TUI to help you debug your cluster.

## Features

- [ ] tui entry
- [ ] kong & bubbletea
- [ ] shortcuts, or interactive commands for most common tasks
- [ ] multiple cluster overview
- [ ] k8s utils (logs, top, getxxx)
- [ ] yaml export
- [ ] fast deploy debugging service
- [ ] custom plugins
- [ ] prefer settings
