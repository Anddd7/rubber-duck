# Chatbot 9527

## Introduction

Local-first chatbot stack for personal usage, focused on self-hosted runtime instead of Kubernetes manifests.

## Use cases

- Run chatbot services on local machine with Docker
- Keep private/local data and model workflow for experiments

## How to use

This app is intentionally outside `kustomize/` because it targets local Docker runtime.
